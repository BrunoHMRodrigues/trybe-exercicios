## O que é?
 CSS (Cascading Style Sheets ou Folha de Estilo em Cascatas em português) é uma linguagem de marcação utilizada para estilizar os conteúdos de linguagens de marcação como o HTML.

 ## Práticas


<style> <style> - Tag utilizada na tag <head> e é onde você informará as propriedades das tags desejadas

 Importante: é possível usar o style dentro de tags, mas não é uma prática recomendada.


## Alguns comandos

// _ - espaço onde informará o valor ou dado desejado

 font-family: _;
 font-style: _;
 font-size: _;
 font-weight: _;

 background-color: _;

 text-align: ;
 text-decoration: _;
 
 line-height: _; 
 
 ## Propriedades
 
 class="<nome da classe>" //Usada como o id, entretanto ela é utilizada quando você quer se referir a um grupo e não uma única. Ex: Você quer que uma mesma estilizacção seja realizada para várias tags diferentes. Uma das maneiras de fazê-lo é utilizar class
 
 .<nome da classe{} //Maneira de chamar a classe e realizar o que quer que seja escrito dentro dos colchetes-->
